专为新西兰生活打造的极致极简、隐私第一的个人财务记账 App。提供高光深色质感与完全的数据主权。

## 产品优势与核心功能

*   **本地优先与绝对隐私 (Offline-First)**：所有记账数据 100% 存在您本机的浏览器物理沙盒中 (`localStorage`)。无云端服务器、无个人数据收集、无泄漏风险，安全感拉满。
*   **智能新西兰对账单 CSV 导入**：支持一键拖入新西兰主流银行（ANZ、ASB、Westpac、BNZ）导出的 CSV 账单。系统会自动解析数据流、去重，并对 Pak'n Save、AT Hop 等本地商户进行智能分类。
*   **无云端多设备对齐 (去重备份与恢复)**：支持一键导出/导入本地加密 JSON 备份文件。恢复时系统通过指纹比对算法自动去重，即使手机和电脑分别记账后合并，也绝对不会产生重复账目。
*   **极简财务指标分析**：一键切换“周/月/年”周期，动态折算日均开销，并以进度条清晰展示占比最高的消费品类。

## 不易察觉的精致细节 (Tips)

*   **PC 鼠标左右拖动**：在电脑浏览器上，您可以通过鼠标左键直接按住并左右拖拽横向的“交易分类栏”，获得如移动端手势滑屏般的流畅体验。
*   **共享空间搜索框**：点击检索 🔍，左侧的 Add 和上传按钮会向左平滑淡出隐藏，搜索框在同排横向以弹簧动效向左侧拉伸展开，空间利用率极高。
*   **重合双国旗徽标**：当录入或导入非纽币（如 CNY / USD）的交易时，左侧标志自动呈现为“本币与外币左右交叠的重合双国旗”（如 🇨🇳 与 🇳🇿 左右重合），直观展示汇率换算关系。
